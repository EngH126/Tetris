# 大一程序设计基础大作业：俄罗斯方块游戏
## 游戏亮点
1.在基础玩法之上增加盲打模式，即不显示已经下落的方块，增加游戏进阶的挑战性
2.可以按快、中、慢调节方块的下落速率，不同的下落速率对应不同的得分倍率
3.主界面的排行榜会自动记录游玩历史前三的得分，排行榜可以重置
4.游戏以任天堂FC机版本的音乐作为背景音乐，设置界面可以调整音量大小

## 实现思路
游戏核心机制实现是通过一个二维数组来存储已下落方块的信息，若该位置已有方块则固定，若满行则消除，若超出窗体则结束；通过srand函数生成随机数，以此对应随机生成的方块种类。
